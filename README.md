# Hornet Helpers
*CSC 131 Group Website Project - A theoretical SacState event organizer*

Hornet Helpers provides a centralized and accessible, student-centered platform connecting prospective student volunteers to relevant campus-based and local organizations/opportunities. This will help to foster a connected Sac State community ecosystem through mutually rewarding service opportunities. Hornet Helpers achieves this by streamlining the overall process of searching, signing up for, and keeping track of students’ community involvement, while simultaneously acting as an easy go-to site for university and local area volunteer-seekers to post calls, as well as recruit, vet, and connect with student volunteers according to their unique needs.

## Tech Stack
- **Frontend:** HTML, CSS, Javascript
- **Backend:** Python with Flask framework and MySQL connector
- **Database:** MySQL
